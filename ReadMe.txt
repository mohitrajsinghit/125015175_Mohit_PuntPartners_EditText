Project README

Project Name: React Text Customization App

Description :- 
This is a React application that allows users to customize text with various font families, font variants (e.g., Bold, Italic), and italic options. The app also features auto-save functionality that retains the user's customizations using local storage, ensuring that the customizations are preserved across page reloads.

Features :- 

Select font family from a list of available options.
Choose font variants such as different weights and italic styles.
Toggle italic style if available for the selected font.
Auto-save customizations to local storage.
Reset to default settings.
